# Adaptive-testing-informatics
Веб-система адаптивного тестування з інформатики для закладів освіти
## Мета проєкту

Метою проєкту є створення веб-системи адаптивного тестування з інформатики, яка забезпечує ефективний, зручний та автоматизований контроль знань учнів.

Система дозволяє формувати навчальні групи, створювати тестування, проходити адаптивні тести зі зміною складності запитань, аналізувати результати та здійснювати персоналізоване оцінювання рівня підготовки учнів.

## Основні функціональні можливості

### Авторизація та аутентифікація

- Реєстрація нових користувачів;
- Вхід до системи за електронною поштою та паролем;
- Розмежування ролей користувачів:
  - Адміністратор
  - Вчитель
  - Учень

### Адміністратор

- Управління користувачами;
- Управління предметами та темами;
- Формування банку тестових запитань;
- Створення, редагування та видалення питань;
- Фільтрація питань за темою та складністю;
- Перегляд статистики системи.

### Вчитель

- Створення навчальних груп;
- Генерація коду запрошення для учнів;
- Створення та редагування тестів;
- Налаштування часу проходження та кількості спроб;
- Вибір тем для тестування;
- Перевірка відкритих відповідей;
- Аналіз результатів учнів.

### Учень

- Приєднання до групи за кодом;
- Проходження тестування;
- Перегляд історії тестів;
- Перегляд результатів проходження.

---

## Адаптивне тестування

Система реалізує механізм адаптивного тестування, у якому складність наступного питання автоматично змінюється залежно від правильності попередніх відповідей учня.

- тестування починається з легкого рівня;
- після правильної відповіді складність може підвищуватись;
- після неправильної — знижуватись;
- оцінювання враховує рівень складності запитань.
